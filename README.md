# Just_Dodge_it

This project demonstrates that Java, even without a dedicated game engine, is capable of handling real tine rendering, physics simulation, detecting collisions, particle effects and persistent data storage. It acts as a practical example of applying object oriented programming concepts to a real project.

**Features**
Clean HUD: Displays Score, High Score, Hull Health, Nitro, and Fuel bars.

Resource Management:

Fuel: Depletes at -3/sec; collect Orange Orbs (+25) to stay alive.

Health: Regenerates at +1.8 HP/sec; collect Green Orbs for rapid recovery.

Dynamic Difficulty: Game speed increases and opponents spawn in pairs (blocking two lanes) as score passes 1500+.

Contextual Game Over: Smart UI triggers a restart screen showing either "WRECKED" or "OUT OF FUEL" based on how you lost.
